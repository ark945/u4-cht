# 術語對照與譯名政策:精訊官方手冊 vs 本專案

> 來源:精訊資訊《創世紀IV》官方繁中手冊(見 [玩家手冊.md](玩家手冊.md) /
> [魔法之書.md](魔法之書.md))。

## 譯名政策:混合採用,**不完全沿用精訊**

本專案**不**全盤套用 1985 精訊官方譯名,而採「**混合**」策略:

- **採精訊**:當官方譯名較完整、貼合原意、或更具年代記憶價值時(例:Dagger 短劍、
  Flaming Oil 焚油、Mace 釘頭鎚、Plate 鎧甲、Fighter 鬥士、Tinker 工匠、Shepherd 牧羊人)。
- **保留現譯**:當官方舊譯今日讀來生硬、易誤解、或現代標準譯法更直觀時
  (例:Paladin **聖騎士**〔非「武士」,避免與 samurai 混淆〕、Ranger **遊俠**〔非「流浪者」,
  後者偏負面〕、Crossbow **弩**〔非「弓箭槍」〕、Staff **法杖**、Sword **長劍**、Sling **投石索**)。
- **數值面**完全忠於原版:武器傷害 / 防具防禦 / 法術 MP 經比對與原版 `AVATAR.EXE`
  **逐位元組相同**;法術試劑組合與《魔法之書》一致。

> 為何不完全採精訊:精訊譯本是 1985 年的時代產物,部分用詞(武士=Paladin、
> 流浪者=Ranger、弓箭槍=Crossbow)在今日語感下偏離原意或易生混淆;本專案以
> 「**正確性 / 直觀 > 純懷舊**」為原則,只在官方譯名同等或更佳時採用,其餘保留現代譯法。
> 八德與三原則沿用台灣《創世紀聖者之書》體系,與精訊、與 u6-cht 一致。
